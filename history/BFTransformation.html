Create a SPARQL query that displays the author and date created

Sure, here's a SPARQL query that displays the author and date created of the BIBFRAME Work Resource:

ruby
PREFIX bf: <http://id.loc.gov/ontologies/bibframe/>
PREFIX dct: <http://purl.org/dc/terms/>

SELECT ?author ?created
WHERE {
  <http://example.org/pots> a bf:Work ;
                           bf:author ?author ;
                           dct:created ?created .
}

In this SPARQL query, we first define the prefixes bf and dct to abbreviate the URIs of the BIBFRAME and Dublin Core Terms vocabularies, respectively. The query selects two variables, ?author and ?created, and retrieves the values for these variables for any resource that is a bf:Work, has an bf:author, and has a dct:created date. The URI of the BIBFRAME Work Resource in this case is http://example.org/pots.
